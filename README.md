# Liatrio Theme for VS Code

A color theme inspired by [liatrio.com](https://www.liatrio.com) branding, featuring the signature green accents in both dark and light variants.

## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **Liatrio Theme**
3. Click **Install**
4. Open **Command Palette** (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Select **Preferences: Color Theme**
6. Choose **Liatrio Dark** or **Liatrio Light**
